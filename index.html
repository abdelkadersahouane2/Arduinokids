مـاهي الأردويـنو

الأردوينو هو منصة إلكترونية مفتوحة المصدر تتكون من جهاز لوحة تطوير صغيرة ومجموعة من البرمجيات (IDE) المصممة لتسهيل البرمجة والتحكم في الإلكترونيات. تأسست الأردوينو في 2005 من قبل مجموعة من الباحثين الإيطاليين بقيادة ماسيمو بانزي، وكان الهدف الأساسي منها هو توفير وسيلة بسيطة ومرنة للمصممين والفنانين والطلاب والهواة لبناء مشاريع إلكترونية بدون الحاجة إلى خلفية واسعة في الإلكترونيات.

شعبية الأردينو

الأردوينو حقق شعبية كبيرة بفضل تصميمه السهل والمتاح، وكونه مفتوح المصدر، مما يعني أن أي شخص يمكنه تطوير وتحسين المنصة. إن توفر الكثير من الوثائق التعليمية، وأمثلة المشاريع، ومجتمع واسع من المستخدمين، يجعل الأردوينو الخيار الأول للعديد من المبتدئين والمحترفين على حد سواء في مجال الإلكترونيات.

يستخدم الأردينو في مجالات عدة وله أنواع متعددة تختلف حسب استخداماتها

أنــواع الأرديــنو

  • Arduino UNO

    أكثر الأنواع شيوعاً، يعتمد على متحكم ATmega328، ويستخدم غالبًا في المشاريع المبتدئة والمتوسطة.

  • Arduino Mega

    يتميز بقدرات أكبر وعدد دبابيس أكثر، ويستخدم في المشاريع التي تتطلب تفاعلات مع عدد كبير من المدخلات والمخرجات، مثل الروبوتات.

  • Arduino Nano

    صغير الحجم ويعتمد على نفس المتحكم في الأردوينو أونو، يستخدم بشكل أساسي في المشاريع التي تتطلب الحجم الصغير.

  • Arduino Leonardo

    يحتوي على متحكم ATmega32u4 الذي يمكنه التفاعل مع الكمبيوتر كأداة إدخال مباشرة، مما يسمح بإنشاء أجهزة مثل لوحات المفاتيح المخصصة أو الفأرات المخصصة.

  • Arduino MKR Series

    سلسلة متخصصة تضم خيارات اتصال مثل الواي فاي والبلوتوث والاتصال الخلوي، مما يجعلها مناسبة لإنترنت الأشياء (IoT).

مجـــالات استخـدام الأردويـنو

  • التعليم

    الأردوينو هو أداة تعليمية شائعة يستخدمها الطلاب والمدرسون في المدارس والجامعات لفهم المبادئ الأساسية في الإلكترونيات والبرمجة.

    • الروبوتات

      يستخدم في العديد من المشاريع الروبوتية، مثل روبوتات تتبع الخطوط، والروبوتات المتحركة، والروبوتات التي تتفاعل مع البيئة.

      • أنترنت الأشياء (Lot)

        يُستخدم في بناء مشاريع ذكية متصلة بالشبكة، مثل المنازل الذكية، أنظمة الري الآلية، ومستشعرات البيئات الذكية.

        • الفن التفاعلي

          يستخدم الأردوينو بشكل شائع من قبل الفنانين لخلق تجارب تفاعلية، حيث يمكنهم برمجته للاستجابة إلى أصوات، حركات، أو لمسات.

          • التحكم في الأجهزة

            يمكن استخدام الأردوينو للتحكم في تشغيل وإطفاء الأجهزة الكهربائية والإلكترونية مثل الأضواء، المحركات، وأنظمة التبريد والتدفئة.

            بعض الأكواد الأساسية في الأردويـنو

            • كود تشغيل وإطفاء LED باستخدام زر

              هذا الكود يقوم بتشغيل وإطفاء LED باستخدام زر. عند الضغط على الزر، يضيء LED، وعند الضغط مرة أخرى ينطفئ. 

              
                        int ledPin = 13;  // تحديد دبوس الـ LED
                        int buttonPin = 2; // تحديد دبوس الزر
                        int buttonState = 0;  // حالة الزر
                        
                        void setup() {
                          pinMode(ledPin, OUTPUT);    // تعيين دبوس LED كخرج
                          pinMode(buttonPin, INPUT);   // تعيين دبوس الزر كمدخل
                        }
                        
                        void loop() {
                          buttonState = digitalRead(buttonPin); // قراءة حالة الزر
                        
                          if (buttonState == HIGH) {   // إذا كان الزر مضغوطًا
                            digitalWrite(ledPin, HIGH);  // تشغيل LED
                          } else {
                            digitalWrite(ledPin, LOW);   // إطفاء LED
                          }
                        }

              الشرح :هذا البرنامج يقرأ حالة الزر. عندما يكون الزر في حالة ضغط، يتم تشغيل الـ LED، وعندما يتم تحرير الزر، يتم إطفاؤه.

            • كود قياس درجة الحرارة باستخدام مستشعر حرارة

              هذا الكود يستخدم مستشعر الحرارة LM35 لقياس درجة الحرارة وعرضها على الشاشة التسلسلية. 

              
                        int tempPin = A0; // دبوس مستشعر الحرارة

                        void setup() {
                          Serial.begin(9600);  // بدء الاتصال التسلسلي
                        }
                        
                        void loop() {
                          int reading = analogRead(tempPin); // قراءة قيمة المستشعر
                          float voltage = reading * (5.0 / 1023.0);  // تحويل القراءة إلى فولت
                          float temperatureC = voltage * 100;  // حساب درجة الحرارة
                        
                          Serial.print("Temperature: ");
                          Serial.print(temperatureC); 
                          Serial.println(" C");
                        
                          delay(1000);  // تأخير لمدة ثانية
                        }

              الشرح :هنا، يقوم الكود بقراءة درجة الحرارة من مستشعر LM35 كل ثانية، ويحولها إلى درجة مئوية، ثم يعرضها على الشاشة التسلسلية.

            • كود تشغيل محرك Servo بزاوية محددة

              هذا الكود يتحكم في محرك Servo لجعله يتحرك إلى زاوية معينة. 

              
                        #include 

                            Servo myservo;  // إنشاء كائن servo
                            int pos = 0;    // موقع السيرفو بالدرجة
                            
                            void setup() {
                              myservo.attach(9);  // ربط السيرفو بالدبوس 9
                            }
                            
                            void loop() {
                              for (pos = 0; pos <= 180; pos += 1) { // التحرك من 0 إلى 180 درجة
                                myservo.write(pos);                
                                delay(15);                       
                              }
                              for (pos = 180; pos >= 0; pos -= 1) { // التحرك من 180 إلى 0 درجة
                                myservo.write(pos);              
                                delay(15);                       
                              }
                            }

              الشرح :يتحرك هذا الكود بمحرك السيرفو من زاوية 0 إلى 180 درجة ببطء، ثم يعيده إلى زاوية 0، ويكرر هذه العملية بشكل مستمر.

            • كود التحكم في LED باستخدام مستشعر ضوء

              يتم تشغيل وإطفاء LED اعتمادًا على شدة الضوء المقاسة بمستشعر ضوء LDR (مقاومة ضوئية). 

              
                        int ledPin = 13;  // دبوس الـ LED
                        int ldrPin = A0;  // دبوس المستشعر الضوئي
                        int threshold = 500; // عتبة الضوء لتشغيل LED
                        
                        void setup() {
                          pinMode(ledPin, OUTPUT);
                          Serial.begin(9600);  
                        }
                        
                        void loop() {
                          int lightLevel = analogRead(ldrPin); // قراءة مستوى الضوء
                          Serial.println(lightLevel); 
                        
                          if (lightLevel < threshold) {
                            digitalWrite(ledPin, HIGH);  // تشغيل LED إذا كان الضوء منخفضًا
                          } else {
                            digitalWrite(ledPin, LOW);   // إطفاء LED إذا كان الضوء عاليًا
                          }
                          
                          delay(500); // تأخير نصف ثانية
                        }

              الشرح :يقرأ هذا الكود مستوى الضوء من المستشعر الضوئي، ويقوم بتشغيل LED إذا كانت شدة الضوء منخفضة، وإطفائه إذا كانت شدة الضوء مرتفعة.

            • كود إرسال بيانات عبر البلوتوث

              يمكن استخدام هذا الكود لإرسال بيانات عبر البلوتوث باستخدام وحدة HC-05 أو HC-06. 

              
                        #include 

                            SoftwareSerial BT(10, 11); // تعيين دبابيس RX وTX للبلوتوث
                            
                            void setup() {
                              Serial.begin(9600);      // بدء الاتصال التسلسلي
                              BT.begin(9600);          // بدء اتصال البلوتوث
                            }
                            
                            void loop() {
                              if (BT.available()) {     // إذا كانت هناك بيانات قادمة من البلوتوث
                                char data = BT.read();  // قراءة البيانات
                                Serial.print("Received: ");
                                Serial.println(data);    // عرض البيانات المستقبلة على الشاشة
                              }
                            
                              if (Serial.available()) {  // إذا كانت هناك بيانات قادمة من التسلسلي
                                char data = Serial.read(); 
                                BT.write(data);         // إرسال البيانات عبر البلوتوث
                              }
                            }

              الشرح :يقوم هذا البرنامج بقراءة البيانات من البلوتوث وعرضها على الشاشة التسلسلية، وكذلك إرسال البيانات من التسلسلي إلى البلوتوث، مما يتيح الاتصال بين الأردوينو وجهاز البلوتوث المتصل.

            القطع الأكثــر شيـوعا في مجموعة Arduino starter kit

            • لوحة Arduino UNO

              الوصف: تعتبر لوحة التحكم الأساسية في المجموعة، تعمل كمتحكم رئيسي لبقية القطع.

              الوظيفة: يتم برمجتها لتنفيذ أوامر محددة، والتفاعل مع مختلف القطع المتصلة بها، وتتحكم في تدفق البيانات.

            • مقاومات كهربائية (Resistors)

              الوصف: قطع صغيرة تحد من تدفق التيار الكهربائي، ويتم تمييز قيمها بألوان عليها.

              الوظيفة: تساعد على حماية القطع الأخرى مثل الـ LED من التلف، وتضمن أن الكمية المناسبة من التيار تصل للدوائر المختلفة.

            • LEDs (الثنائيات الضوئية)

              الوصف: وحدات ضوئية صغيرة تعمل عند مرور التيار الكهربائي.

              الوظيفة: تُستخدم كإشارات ضوئية، وغالبًا ما توضع في المشاريع كإشارة لإضاءة أو إشعار مرئي.

            • الأسلاك (Jumper Wires)

              الوصف: أسلاك صغيرة تُستخدم لربط مكونات الدائرة على لوحة التجارب بدون لحام.

              الوظيفة: تُستخدم لتمرير الإشارات والطاقة بين مكونات الدائرة، مثل توصيل المستشعرات، والمحركات، والمكونات الإلكترونية الأخرى بلوحة الأردوينو.

            • زر الضغط (Push Button)

              الوصف: زر بسيط يولد اتصالًا أو يقطعه عند الضغط عليه.

              الوظيفة: يُستخدم كمفتاح تشغيل أو تعطيل في الدوائر، حيث يسمح بالتحكم اليدوي بالدوائر الإلكترونية.

            • مقاومة متغيرة (Potentiometer)

              الوصف: مقاومة ذات قيمة متغيرة يمكن تعديلها باستخدام مقبض أو قرص.

              الوظيفة: تُستخدم للتحكم في شدة الإشارة أو التيار، مثل التحكم في شدة الضوء أو الصوت.

            • حساس درجة الحرارة (Temperature Sensor)

              الوصف: حساس يقيس درجة الحرارة المحيطة ويحوّلها إلى قيمة كهربائية.

              الوظيفة: يستخدم لقراءة درجة الحرارة، ويُستخدم في مشاريع قياس والتحكم بالبيئة.

            • حساس الضوء (Photoresistor)

              الوصف: مقاومة تتغير قيمتها حسب كمية الضوء الساقط عليها.

              الوظيفة: يُستخدم لاستشعار شدة الإضاءة، مثل إطفاء الأنوار تلقائيًا في النهار وتشغيلها في الليل.

            • محرك السيرفو (Servo Motor)

              الوصف: محرك صغير يعمل على تحديد زوايا معينة بدقة.

              الوظيفة: يُستخدم في مشاريع تتطلب التحكم في زاوية محددة، مثل التحكم في الأبواب، أو الروبوتات.

            • محرك كهربائي (DC Motor)

              الوصف: محرك بسيط يعمل بالتيار المستمر.

              الوظيفة: يُستخدم لتحريك الأشياء، ويعتبر أساسيًا في بناء الروبوتات البسيطة.

            • شاشة عرض (LCD)

              الوصف: شاشة صغيرة تعرض النصوص والبيانات.

              الوظيفة: تُستخدم لعرض الرسائل أو المعلومات، مثل درجة الحرارة أو البيانات المستقبلة من الحساسات.

            • مقاومات حرارية (Thermistor)

              الوصف: نوع من المقاومات تتغير قيمتها حسب درجة الحرارة.

              الوظيفة: تُستخدم في قياس التغير في درجة الحرارة، مثل حساسات الحرارة في التبريد والتدفئة.

            • لوحة تجارب (Breadboard)

              الوصف: لوحة مؤقتة لتجميع وتوصيل الدوائر دون الحاجة إلى لحام.

              الوظيفة: تتيح لك تجربة توصيلات الدوائر وتركيب القطع بسهولة وسرعة قبل تثبيتها نهائيًا.

            • متحسس الحركة (PIR Sensor)

              الوصف: حساس يستشعر الأشعة تحت الحمراء للكائنات.

              الوظيفة: يستخدم لاستشعار الحركة، ويُستخدم غالبًا في أنظمة الأمان والتحكم التلقائي.

            • حساس الرطوبة والحرارة (DHT11)

              الوصف: حساس يقيس كل من الرطوبة ودرجة الحرارة.

              الوظيفة: يُستخدم في المشاريع البيئية، حيث يعطي بيانات عن درجة الحرارة ونسبة الرطوبة في المكان.

            • مستشعر الصوت (Sound Sensor)

              الوصف: حساس يستجيب للأصوات حوله.

              الوظيفة: يُستخدم لالتقاط الأصوات أو قياس شدتها، مثل استشعار الصوت في المشاريع التفاعلية.

            • وحدة البلوتوث (Bluetooth Module)

              الوصف: وحدة لاسلكية للتواصل مع الأجهزة الأخرى عبر البلوتوث.

              الوظيفة: تُستخدم في التحكم عن بُعد بالأجهزة، مثل التحكم بالروبوتات أو جمع البيانات بشكل لاسلكي.

            • مخرج الصوت (Buzzer)

              الوصف: جهاز صغير يصدر أصواتًا عند تمرير تيار كهربائي خلاله.

              الوظيفة: يُستخدم لإصدار التنبيهات الصوتية في المشاريع.

            • مقياس الرطوبة (Humidity Sensor)

              الوصف: حساس يقيس مستوى الرطوبة في البيئة المحيطة.

              الوظيفة: يُستخدم في مراقبة مستوى الرطوبة، ويعتبر مفيدًا في تطبيقات المناخ والزراعة.
